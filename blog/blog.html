Prompt

This page will be a place to share experiments—scribbles, snippets, and half-formed ideas—mostly inspired by Songho’s legendary note compilation. It may not be polished, but hopefully it will be coherent.

I built this page to push myself to write more, something I haven’t done properly in some time. It also carves out a small corner of the web for myself. Writing and posting here is a way to shift from consuming to creating.

"italktorobots, because sometimes what we say has so much noise, only the computer can parse it."

The name just sounded cool, something a rogue cyborg in Ghost in the Shell might say to the Major.

Lately, I’ve been looking inward. Programming, science, math, the tools that have shaped the world have also shaped us. This page is a reflection on that.
